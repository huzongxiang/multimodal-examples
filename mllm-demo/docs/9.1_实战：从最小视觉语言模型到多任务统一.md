# 9.1 VLM实战：从最小视觉语言模型到多任务统一

本案例围绕一个最小视觉语言系统展开。系统始终保持同一套 MiniVLM 主体结构：视觉编码器提取图像特征，映射层将视觉特征投影到语言模型隐藏空间，语言模型则在统一生成接口上完成描述、检测与问答。与为不同任务分别设计独立任务头的做法相比，这一路线更强调“任务文本化”与“接口统一化”，因而更适合用来观察一个最小多模态生成系统是如何被逐步搭建出来的。

整组实验按四个阶段推进。第一阶段用图像描述建立最小跨模态连接，验证图像特征是否已经能够驱动文本生成；第二阶段把目标检测改写为 JSON 风格的结构化生成任务，考察统一接口是否能够承担更明确的视觉预测；第三阶段引入视觉问答，检验模型在图像与文本双条件下是否形成回答能力；第四阶段再把描述、检测与问答混合到同一训练过程中，观察同一组参数能否在不同提示下完成任务切换。四个阶段连在一起，构成了一条从最小跨模态连接到多任务统一接口初步形成的能力演进路径。
